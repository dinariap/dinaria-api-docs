# FX Handling

> **Coming soon.** FX conversion between currencies is not yet available. All payouts today must be in the same currency as your merchant balance — ARS for Argentine merchants, BRL for Brazilian merchants.

This page will document FX quote and conversion endpoints once the feature is released. If you have a cross-currency use case, contact support.
