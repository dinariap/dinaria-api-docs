# Payment API â€” Brazil (PIX)

Welcome to the Dinaria API documentation for Brazil.

This portal contains guides and references to help you integrate PIX payments quickly and safely.

## What you can do with this API

- Create payments using PIX (instant, 24/7)
- Track payment status changes
- Send payouts to recipients via PIX keys
- Receive asynchronous updates via webhooks

## PIX keys

Recipients (and sometimes payers) are identified by a PIX key. Supported types:

| Type    | Example              |
|---------|----------------------|
| CPF     | `12345678901`        |
| CNPJ    | `12345678000199`     |
| Email   | `user@example.com`   |
| Phone   | `+5511999999999`     |
| Random  | Alphanumeric key     |

## Key identifiers

- `transactionId`: ID generated by our platform (primary identifier)
- `externalId`: ID you provide (your order/checkout reference)
