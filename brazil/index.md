# Dinaria API — Brazil (PIX)

The Dinaria API for Brazil enables PIX instant payments and transfers.

PIX is Brazil's instant payment system (run by the central bank). Payments and
payouts settle in seconds, 24/7. Recipients are identified by a PIX key: CPF,
email, phone number, or random alphanumeric key.
