# Data formats (ISO standards)

## country
ISO 3166-1 alpha-2 (e.g. `US`, `UY`, `MX`).

## state / region
Recommended: ISO 3166-2 (e.g. `US-CA`, `US-NY`).
If the merchant does not know the ISO code, send a commonly used region name.

## city
Free-text string (no universally practical standard for city codes).

## currency
ISO 4217 (e.g. `USD`, `EUR`).
